# RBAC权限控制
用户-----角色-----权限 用户和权限间接相连（将用户分配角色  权限分配给角色）

                                      RBAC模块功能
                                      
           用户管理                       角色管理                     权限管理
         1、用户列表                     1、角色列表                  1、权限列表
         2、添加用户                     2、添加角色                  2、添加权限
         3、编辑用户                     3、编辑角色                  3、编辑权限
         4、设置角色                     4、设置权限
